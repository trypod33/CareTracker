# CareTracker

CareTracker is a local-first Android caregiver support app scaffold built in Kotlin.

## Features included
- Multi-person tracking
- Appointments and recurring calendar entries
- Medication management with pill inventory and refill reminder foundation
- Care contacts (doctors, nurse practitioners, therapists)
- Health tracking for blood sugar, water, blood pressure, heart rate, sleep, and steps
- Habit tracking for caregiver and cared-for individuals
- Gentle caregiver encouragement messages
- Local Room database foundation

## Notes
This is a strong starter scaffold intended for opening in CodeAssist / Android Studio and continuing development.
Some advanced pieces are stubbed and ready for expansion, including:
- repository layer
- ViewModels
- RecyclerView adapters
- Room dependency injection module
- chart screens
- WorkManager scheduling for reminders
- recurring appointment expansion logic

## Open
Open the CareTracker folder in CodeAssist or Android Studio and sync Gradle.
